# point_counter
Simple point counter to count how many points someone/a team has.

# Features
- Individual point counter
- Team point counter
- Renameable players
- Renameable teams
- Ability to change team colours
- Ability to change player colours
- Cool animation when taking away points
- Cool animation when giving points
- Fullscreen button
- Homepage button
- Text size able to be changed up to 300%
- Text wrapping in individual play
- Discreet settings menu
- Text is centered
